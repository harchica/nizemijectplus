# nizemiject
二ゼミプロジェクト(名称未定)

## 概要
SNSに書き込めないような嬉しいことや悲しいこと(むしゃくしゃしたこと)を
数字として可視化、もし内容も書き出したければ送信して
あとで振り返るというアプリケーションにする
